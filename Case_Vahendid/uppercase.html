Ülemise taseme CASE vahendid(upper CASE-tools);
        Toetavad analüüsi ja projekteerimist.
        Peamiselt on nad kasutusel kasutajanõuete analüüsimisel ja dokumenteerimisel. 
        Nad on ennekõike mõeldud visualiseerimiseks, erinevate skeemide koostamiseks ja ka dokumentatsiooni genereerimiseks.
        Nad toetavad traditsiooniliste diagrammikeelte kasutamist (olem-seos diagrammid, andmemudelid, UML-skeemid, jne).

        Upper CASE ehk backend tööriistad;
        C++ - Hypertext Markup Language
         Üldotstarbeline staatiliste andmetüüpidega multifunktsionaalne programmeerimiskeel, 
                                                                            mis toetab abstraheerimist, polümorfismi, protseduraalset, objektorienteeritud ja üldistavat programmeerimist.
        Python - Cascading Style Sheets
         Üldotstarbeline interpreteeritav programmeerimiskeel, mida algselt arendati skriptimiskeeleks.
        Node.js - JavaScript
         Avatud lähtekoodiga mitmeplatvormne JavaScripti käitussüsteem, mille abil saab arendada mitmesuguseid rakendusi.
